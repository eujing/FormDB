1) Make sure MySQL is installed (version used: 5.5.35)
2) Run create.sql to create the database
3) Fill in dbCredentials.xml with database login credentials

On insertion of a new kind of form
1) Press "Browse" and select pdf form to insert into database
2) Select "New..." under Table Name
3) Fill in name of new table for that kind of form
4) Press "Submit"

On insertion of an existing kind of form
1) Press "Browse" and select pdf form to insert into database
2) Select corresponding table name
4) Press "Submit"
